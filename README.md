### Requirements
- Docker

### Installation
```bash
git clone ...
docker-compose build
docker-compose up
```

### Common Commands

```bash
  # Migrate:              
  truffle migrate
  
  # Test contracts:       
  truffle test
  
  # Test dapp:            
  npm test
  
  # Run dev server:       
  npm run start
  
  # Build for production: 
  npm run build
```
